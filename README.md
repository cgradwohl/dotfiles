### GNU Stow Options

-v - verbose -t - target directory

### GNU Stow Process

1. cd into dotfiles repo
2. run `stow -v -t ~ <package_name>`

### Diagnostic Keybindings

- `gl` - Show diagnostic in floating window
- `]d` - Jump to next diagnostic
- `[d` - Jump to previous diagnostic
- `<leader>xx` - Toggle Trouble diagnostics list
- `<leader>xX` - Buffer-only diagnostics (Trouble)
